A **cluster** is a group of machines that ==work together and appear as a single logical system==. Clusters provide scalability (more machines → more capacity) and fault tolerance (if one node fails others continue).

Why use clusters
- Scale horizontally (add nodes)
- Improve availability and throughput
- Support distributed storage and compute

Common cluster types
- ==Compute clusters== (Spark, HPC)
- ==Storage clusters== (HDFS, Ceph)
- Database clusters (Cassandra, MySQL group replication)
