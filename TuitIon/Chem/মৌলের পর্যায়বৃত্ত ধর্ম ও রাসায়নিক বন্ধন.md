## ৩.১ — মৌলের শ্রেণিবিভাগ (Block Classification) 🔴

### আধুনিক পর্যায় সূত্র

> **"মৌলসমূহের ভৌত ও রাসায়নিক ধর্মাবলি এদের পারমাণবিক সংখ্যা বৃদ্ধির সাথে পর্যায়ক্রমে আবর্তিত হয়।"**

### ব্লক ভাগ

|ব্লক|সর্বশেষ ইলেকট্রন|উদাহরণ|বৈশিষ্ট্য|
|---|---|---|---|
|**s-ব্লক**|ns¹⁻²|H, He, গ্রুপ ১, গ্রুপ ২|সক্রিয় ধাতু|
|**p-ব্লক**|np¹⁻⁶|গ্রুপ ১৩–১৮|ধাতু + অধাতু + অর্ধধাতু|
|**d-ব্লক**|(n–1)d¹⁻¹⁰ ns¹⁻²|Sc থেকে Zn (Th সহ ৮৫টি)|অবস্থান্তর ধাতু|
|**f-ব্লক**|(n–2)f¹⁻¹⁴|Lanthanides, Actinides|গ্রুপ-৩ এর অন্তর্ভুক্ত|

### ইলেকট্রন বিন্যাস থেকে অবস্থান নির্ণয়

- **পর্যায় সংখ্যা** = সর্বোচ্চ প্রধান কোয়ান্টাম সংখ্যা **n**
- **গ্রুপ সংখ্যা:**
    - s-ব্লক: ns এর মোট e⁻ = গ্রুপ নং
    - p-ব্লক: (10 + ns² + np^x) = গ্রুপ নং
    - d-ব্লক: (n–1)d + ns এর মোট e⁻ = গ্রুপ নং

```
উদাহরণ:
Na(11) → 1s² 2s² 2p⁶ [3s¹]  → পর্যায়-৩, গ্রুপ-১
Cl(17) → 1s² 2s² 2p⁶ [3s² 3p⁵] → পর্যায়-৩, গ্রুপ-১৭
Cr(24) → 1s² 2s² 2p⁶ 3s² 3p⁶ [3d⁵ 4s¹] → পর্যায়-৪, গ্রুপ-৬
Cu(29) → 1s² 2s² 2p⁶ 3s² 3p⁶ [3d¹⁰ 4s¹] → পর্যায়-৪, গ্রুপ-১১
```

> ⚠️ **Cr ও Cu এর e⁻ বিন্যাস ব্যতিক্রম** — অর্ধপূর্ণ/পূর্ণ d অরবিটাল বেশি স্থিতিশীল

## ৩.২ — s-ব্লক মৌলের ধর্ম 🔴

### গ্রুপ-১ (ক্ষার ধাতু): Li, Na, K, Rb, Cs, Fr

|ধর্ম|বিবরণ|
|---|---|
|ইলেকট্রন বিন্যাস|ns¹|
|যোজ্যতা|১ (একক ধনাত্মক আয়ন Li⁺, Na⁺ ইত্যাদি)|
|বর্ণ|বর্ণহীন — যৌগও বর্ণহীন (ব্যতিক্রম: K₂CrO₄ হলুদ, KMnO₄ বেগুনি)|
|গলনাঙ্ক|নিচে নামলে কমে — Li 180.5°C → Cs 28.4°C|
|কাটা|ছুরি দিয়ে কাটা যায় (নরম)|
|পানির সাথে|তীব্র বিক্রিয়া → MOH + H₂ গ্যাস|
|অক্সিজেনের সাথে|Li → Li₂O; Na → Na₂O₂; K,Rb,Cs → MO₂|
|আয়নীকরণ শক্তি|গ্রুপে নিচে নামলে কমে|
|ক্যাটায়ন আকার|Li⁺ < Na⁺ < K⁺|

> 🔴 **Be বাদে s-ব্লকের অন্য সব ধাতু আয়নিক যৌগ গঠন করে** 🔴 **Be²⁺ আয়ন কখনো সৃষ্টি হয় না** (আকার খুব ছোট, চার্জ ঘনত্ব বেশি)

### গ্রুপ-২ (মৃৎ-ক্ষার ধাতু): Be, Mg, Ca, Sr, Ba, Ra

|ধর্ম|বিবরণ|
|---|---|
|ইলেকট্রন বিন্যাস|ns²|
|যোজ্যতা|২ (দ্বিধনাত্মক আয়ন Be²⁺, Mg²⁺ ইত্যাদি)|
|গলনাঙ্ক|গ্রুপ-১ এর চেয়ে বেশি (দ্বিধনাত্মক আয়ন = শক্তিশালী বন্ধন)|
|কার্বনেটের স্থায়িত্ব|নিচে নামলে বাড়ে: BeCO₃ 100°C → BaCO₃ 998°C|

## ৩.৩ — p-ব্লক মৌলের ধর্ম 🔴

### গ্রুপ-১৬ হাইড্রাইড (গুরুত্বপূর্ণ!) 🟢

|যৌগ|অবস্থা|কারণ|
|---|---|---|
|**H₂O**|তরল (100°C ফুটনাঙ্ক)|আন্তঃআণবিক H-বন্ধন (প্রতি H₂O তে ৩.৪টি H-বন্ধন)|
|**H₂S**|গ্যাস|দুর্বল ভ্যানডার ওয়ালস বল, H-বন্ধন নেই|
|H₂Se, H₂Te|গ্যাস|ভ্যানডার ওয়ালস বল|

> ⚠️ **OF₂ গঠিত হয় কিন্তু OF₄ ও OF₆ গঠিত হয় না** — O পরমাণুতে কোনো খালি 2d অরবিটাল নেই, তাই যোজ্যতা বাড়াতে পারে না।

### গ্রুপ-১৭ (হ্যালোজেন): F, Cl, Br, I

- পারমাণবিক সংখ্যা বাড়লে সক্রিয়তা **কমে**: F > Cl > Br > I
- তড়িৎ ঋণাত্মকতা: F সর্বোচ্চ (4.0)

### HF এর অস্বাভাবিক আচরণ 🟢

- ফুটনাঙ্ক = 19.5°C (HCl, HBr, HI এর চেয়ে অনেক বেশি)
- কারণ: HF অণুগুলো কঠিন ও তরল অবস্থায় H-বন্ধন দ্বারা যুক্ত থাকে → (HF)ₙ পলিমার গঠন
- তরল HF এ একটি H-বন্ধন থাকে, কিন্তু কঠিন HF এ সর্বোচ্চ ২টি H-বন্ধন থাকতে পারে

## ৩.৪ — d-ব্লক (অবস্থান্তর মৌল) 🔴

### সাধারণ ইলেকট্রন বিন্যাস

```
(n–1)d¹⁻¹⁰ ns¹⁻²     যেখানে n = 4, 5, 6, 7
```

### অবস্থান্তর মৌলের সংজ্ঞা

> যে সব d-ব্লক মৌল অন্তত এমন একটি আয়ন গঠন করে যাদের ইলেকট্রন বিন্যাসে d-অরবিটাল আংশিকভাবে পূর্ণ (d¹–d⁹) থাকে।

> 🔴 **Sc ও Zn অবস্থান্তর মৌল নয়** — Sc³⁺ এ d⁰, Zn²⁺ এ d¹⁰ (সম্পূর্ণ d)

### অবস্থান্তর মৌলের বৈশিষ্ট্য

1. পরিবর্তনশীল জারণ অবস্থা (Variable oxidation states)
2. রঙিন আয়ন ও যৌগ গঠন
3. জটিল যৌগ গঠন করে
4. প্রভাবকরূপে ক্রিয়া করে
5. প্যারা চুম্বকীয় ধর্ম

### অবস্থান্তর ধাতুর রঙিন আয়ন 🟢 (মুখস্থ কর!)

|আয়ন|d-কনফিগ|রং|
|---|---|---|
|Ti³⁺|d¹|বেগুনি|
|V⁴⁺|d¹|নীল|
|V³⁺|d²|সবুজ|
|Cr³⁺|d³|বেগুনি|
|Mn³⁺, Mn²⁺|d⁴, d⁵|বেগুনি, গোলাপি|
|Fe³⁺|d⁵|হলুদ|
|Fe²⁺|d⁶|সবুজ|
|Co²⁺|d⁷|গোলাপি|
|Ni²⁺|d⁸|সবুজ|
|Cu²⁺|d⁹|নীল|
|Zn²⁺|d¹⁰|**বর্ণহীন** (d¹⁰ → রঙ নেই)|

### Cr এর জারণ অবস্থা

```
CrCl₂ (Cr: +2)   CrCl₃ (Cr: +3)   K₂CrO₄ (Cr: +6)   K₂Cr₂O₇ (Cr: +6)
```

### Fe এর জারণ অবস্থা

```
FeO (Fe: +2)   Fe₂O₃ (Cr: +3)
```

### d-ব্লক মৌলের প্রভাবক উদাহরণ 🟢

- **V₂O₅** → SO₂ থেকে SO₃ তৈরিতে প্রভাবক (H₂SO₄ শিল্প)
- **Fe** → N₂ + H₂ থেকে NH₃ তৈরিতে (হাবার পদ্ধতি)
- **Ni(28)** → হাইড্রোজিনেশনে প্রভাবক (উদ্ভিজ্জ তেল → ঘি)

### পারমাণবিক আকার — গুরুত্বপূর্ণ তুলনা

```
Na > Mg > Al > Si > P > S > Cl > Ar  (পর্যায়-৩ এর ক্রম)
Li < Na < K < Rb < Cs              (গ্রুপ-১ এর ক্রম)
```

> আয়নিক ব্যাসার্ধ: **Na⁺ < Mg²⁺ < Al³⁺** (বেশি চার্জ = বেশি আকর্ষণ = ছোট) কিন্তু **Be³⁺ সবচেয়ে ছোট** আয়ন

### আয়নীকরণ শক্তি — ব্যতিক্রম ⚠️

```
সাধারণ ধারা: বাম থেকে ডানে বাড়ে
কিন্তু ব্যতিক্রম:
  Be (4) > B (5)   — Be এ পূর্ণ 2s², B তে 2p¹ (সহজে অপসারিত)
  N (7) > O (8)    — N এ অর্ধপূর্ণ 2p³ (stable), O তে যুগলবদ্ধ 2p (বিকর্ষণ)
```

> 🩷 **মেডিকেল ট্র্যাপ**: "O এর IE, N এর চেয়ে কম কেন?" — কারণ O তে 2p অরবিটালে যুগলবদ্ধ ইলেকট্রন থাকায় বিকর্ষণ বল কাজ করে।

### ইলেকট্রন আসক্তি — ব্যতিক্রম

```
Cl > F  — F এর আকার ছোট বলে বাড়তি ইলেকট্রনের বিকর্ষণ বেশি
N < C   — N এ অর্ধপূর্ণ p orbital (স্থিতিশীল) বলে ইলেকট্রন নিতে চায় না
```

### তড়িৎ ঋণাত্মকতা — Pauling স্কেল 🟢

```
F = 4.0   O = 3.5   N = 3.0   Cl = 3.0
Br = 2.8  C = 2.5   S = 2.5   I = 2.5   H = 2.1
```

> ΔEN দিয়ে বন্ধনের প্রকৃতি:
> 
> - ΔEN > 1.7 → প্রধানত আয়নিক
> - ΔEN 0.4–1.7 → পোলার সমযোজী
> - ΔEN < 0.4 → প্রধানত সমযোজী
> - ΔEN = 0 → অপোলার সমযোজী

### অক্সাইডের ধর্ম

|অক্সাইড প্রকার|উদাহরণ|
|---|---|
|ক্ষারধর্মী|Na₂O, K₂O, CaO, BaO|
|অম্লধর্মী|SO₃, N₂O₅, Cl₂O₇|
|উভধর্মী (Amphoteric)|Al₂O₃, ZnO, SnO₂, PbO₂, BeO, Ga₂O₃|

> 🔴 **BaO সবচেয়ে বেশি ক্ষারীয়, N₂O₅ সবচেয়ে বেশি অম্লীয়** 🔴 **SnO₂ উভধর্মী** — মনে রাখো: Sn হলো গ্রুপ-১৪ এর পঞ্চম মৌল যারা উভধর্মী অক্সাইড গঠন করে

### কর্ণ সম্পর্ক (Diagonal Relationship) 🟢

```
Li ↘ Be ↘ B  ↘ C
Na  Mg  Al  Si
```

- **Li ~ Mg**, **Be ~ Al**, **B ~ Si** — ধর্মে সাদৃশ্য আছে
- কারণ: আয়নিক ব্যাসার্ধ, আয়নীকরণ শক্তি, তড়িৎ ঋণাত্মকতা প্রায় সমান

## ৩.৯–৩.১২ — অরবিটাল সংকরণ 🔴

### সংকরণ তালিকা (অবশ্যই মুখস্থ!) 🟢

|সংকরণ|আকৃতি|বন্ধন কোণ|উদাহরণ|
|---|---|---|---|
|**sp**|রৈখিক (Linear)|180°|BeCl₂, C₂H₂ (acetylene), CO₂|
|**sp²**|সমতলীয় ত্রিভুজাকার|120°|BF₃, BCl₃, C₂H₄ (ethylene), SO₃|
|**sp³**|চতুষ্তলকীয় (Tetrahedral)|109.5°|CH₄, CCl₄, NH₃*, H₂O*|
|**sp³d**|ত্রিকোণাকার দ্বিপিরামিড|90°, 120°|PCl₅, SF₄|
|**sp³d²**|অষ্টতলকীয় (Octahedral)|90°|SF₆, PCl₆⁻|

> *NH₃ — sp³ কিন্তু কোণ 107° (একটি lone pair আছে) *H₂O — sp³ কিন্তু কোণ 104.5° (দুটি lone pair আছে)

### গুরুত্বপূর্ণ সংকরণ বিস্তারিত

#### sp³ → মিথেন (CH₄)

```
C → 2s² 2p² (স্বাভাবিক)
C* → 2s¹ 2p³ (উদ্দীপিত)
sp³ সংকরণ → 4টি সমশক্তির অরবিটাল
বন্ধন কোণ: 109.5°, চতুষ্তলকীয় আকৃতি
```

#### sp² → ইথিন বা ইথিলিন (C₂H₄)

```
C* → 2s¹ 2p_x¹ 2p_y¹ 2p_z¹
sp² সংকরণ: 2s + 2p_x + 2p_y → 3টি সংকর অরবিটাল (একই সমতলে, 120°)
অবশিষ্ট 2p_z → π বন্ধন গঠন করে
ফলে: C–C এ একটি σ + একটি π বন্ধন (double bond)
33.3% s-চরিত্র ও 66.7% p-চরিত্র
```

#### sp → ইথাইন বা অ্যাসিটিলিন (C₂H₂)

```
sp সংকরণ: 2s + 2p_x → 2টি অরবিটাল (180° কোণে)
অবশিষ্ট 2p_y, 2p_z → ২টি π বন্ধন গঠন করে
ফলে: C≡C এ একটি σ + দুটি π বন্ধন (triple bond)
```

### কার্বনের বিভিন্ন রূপ ও সংকরণ 🟢

|পদার্থ|সংকরণ|বৈদ্যুতিক পরিবাহিতা|
|---|---|---|
|হীরা (Diamond)|sp³|অপরিবাহী|
|গ্রাফাইট|sp²|পরিবাহী (মুক্ত π ইলেকট্রন)|
|অ্যালকেন (CₙH₂ₙ₊₂)|sp³|অপরিবাহী|
|অ্যালকিন (CₙH₂ₙ)|sp² (double bond C)|—|
|অ্যালকাইন (CₙH₂ₙ₋₂)|sp (triple bond C)|—|

## ৩.১৩ — সন্নিবেশ সমযোজী বন্ধন (Coordinate Bond) 🔴

> **সংজ্ঞা:** নিঃসঙ্গ ইলেকট্রন যুগল দ্বারা দুটি পরমাণু সমযোজী বন্ধন গঠন করলে, মুক্তজোড় ইলেকট্রন যুক্ত পরমাণুটিকে **দাতা** এবং অপরটিকে **গ্রহীতা** বলে।

### গুরুত্বপূর্ণ উদাহরণ 🟢

```
NH₃ + H⁺ → NH₄⁺   (N দাতা, H⁺ গ্রহীতা)
BF₃ + F⁻ → BF₄⁻   (F⁻ দাতা, B গ্রহীতা)
BF₃ + NH₃ → BF₃·NH₃   (N → B সন্নিবেশ বন্ধন)
H₂O + H⁺ → H₃O⁺   (O দাতা)
```

## ৩.১৫ — ভ্যানডার ওয়ালস বল (Van der Waals Forces) 🔴

### তিন ধরনের আন্তঃআণবিক বল

|বলের নাম|প্রকৃতি|কখন কার্যকর|
|---|---|---|
|**আয়ন-ডাইপোল**|আয়নিক যৌগ + পোলার দ্রাবক|আয়নিক দ্রবীভূতকালে|
|**ডাইপোল-ডাইপোল**|দুটি পোলার অণু|পোলার সমযোজী যৌগে|
|**লন্ডন বিস্তারণ বল**|সব অণুতে (অপোলারেও)|সব পদার্থে|

> 🔴 **বিস্তারণ বল সবচেয়ে দুর্বল** কিন্তু সব ধরনের অণুতেই বিদ্যমান।

### অণুর মধ্যে কার্যকর বল চেনার নিয়ম 🟢

```
HCl     → পোলার; ডাইপোল-ডাইপোল + বিস্তারণ বল (H-bond নেই কারণ Cl যথেষ্ট EN নয়)
CH₃CH₃  → অপোলার; শুধু বিস্তারণ বল
CH₃NH₂  → পোলার; H-বন্ধন + ডাইপোল-ডাইপোল + বিস্তারণ বল
Kr      → অপোলার; শুধু বিস্তারণ বল
```

## ৩.১৬ — হাইড্রোজেন বন্ধন (H-Bond) 🔴

### সংজ্ঞা

> অধিক তড়িৎ ঋণাত্মক পরমাণু **F, O, N** এর সাথে যুক্ত H পরমাণুর মধ্যবর্তী বন্ধন যা অধিকতর পোলার হয়।

### বৈশিষ্ট্য

- শক্তিমাত্রা: **10–40 kJ/mol** (সমযোজী বন্ধনের চেয়ে অনেক দুর্বল)
- **আন্তঃআণবিক H-বন্ধন**: দুটি ভিন্ন অণুর মধ্যে (H₂O, HF, NH₃)
- **অন্তঃআণবিক H-বন্ধন**: একই অণুর মধ্যে (o-nitrophenol)

### H-বন্ধনের কারণে ফুটনাঙ্ক তুলনা 🟢

```
H₂O (100°C) >> H₂S (−60°C)  — H₂O তে H-বন্ধন আছে, H₂S তে নেই
HF (19.5°C) >> HCl (−85°C)  — HF এ H-বন্ধন আছে
NH₃ (−33°C) > PH₃ (−87°C)  — NH₃ এ দুর্বল H-বন্ধন আছে
```

### H-বন্ধনের গুরুত্ব (জীবজগতে) 🟢

- DNA দ্বিসূত্রী গঠন ধরে রাখে H-বন্ধন দ্বারা
- প্রোটিনের secondary structure (α-helix, β-sheet)
- পানির উচ্চ ফুটনাঙ্ক → জীবনের জন্য অপরিহার্য

### AgCl, AgI, HgCl₂, PbCl₂ — রঙ মনে রাখার টেবিল 🟢

```
AgCl  → সাদা     AgI  → হলুদ
HgCl₂ → সাদা    HgI₂ → লাল
PbCl₂ → সাদা    PbI₂ → সোনালি হলুদ
```

## 🧠 Key Formulas & Quick Reference

### অরবিটালে অংশগ্রহণকারী সংখ্যা (H) নির্ণয়ের সূত্র

```
H = ½ [V + X – C + A]
V = যোজ্যতা স্তরের ইলেকট্রন সংখ্যা
X = একযোজী পরমাণু সংখ্যা
C = ক্যাটায়নের চার্জ
A = অ্যানায়নের চার্জ
```

### দ্রুত মনে রাখার সূত্র

```
Single bond  = 1σ
Double bond  = 1σ + 1π
Triple bond  = 1σ + 2π

Bond strength: σ > π
Bond reactivity: π > σ (π বেশি সক্রিয়)
```
