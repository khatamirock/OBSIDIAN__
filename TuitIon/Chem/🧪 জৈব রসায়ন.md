# 📚 সেকশন ২.০ — জৈব রসায়ন পরিচিতি (Introduction)

# ═══════════════════════════════════════════

## 🔥 হাই-ভ্যালু টপিক সতর্কতা

> ⚠️ **এই সেকশন থেকে বেশিরভাগ MCQ আসে।** জৈব যৌগের সংজ্ঞা, প্রাণশক্তি মতবাদ খণ্ডন, এবং জৈব ও অজৈব যৌগের পার্থক্য থেকে প্রতি বছর প্রশ্ন আসে। **ওয়েলার-এর পরীক্ষা মনে রাখা বাধ্যতামূলক।**

## ✅ মূল সংজ্ঞাসমূহ

|বিষয়|সংজ্ঞা|
|---|---|
|**জৈব রসায়ন**|কার্বন শিকল দ্বারা গঠিত যৌগসমূহের রসায়ন (কার্বন যৌগের রসায়ন)|
|**হাইড্রোকার্বন**|কার্বন ও হাইড্রোজেন দ্বারা গঠিত জৈব যৌগ — সকল জৈব যৌগের মূল ভিত্তি|
|**কার্যকরী মূলক**|জৈব যৌগে যে বিশেষ গ্রুপ বা মূলক যৌগের রাসায়নিক ধর্ম নির্ণয় করে|
|**সমগোত্রীয় শ্রেণি**|একই কার্যকরী মূলকবিশিষ্ট যৌগের সিরিজ যেখানে প্রতিটি সদস্য CH₂ দ্বারা পৃথক|

## 📌 প্রাণশক্তি মতবাদ (Vital Force Theory)

- **১৮০৮ সালে** সুইডিশ বিজ্ঞানী **বার্জেলিয়াস** প্রস্তাব করেন → জৈব যৌগ শুধু সজীব উৎস থেকে তৈরি হয়
- **১৮২৮ সালে ওয়েলার (F. Wöhler)** পরীক্ষাগারে অজৈব অ্যামোনিয়াম সায়ানেট থেকে ইউরিয়া তৈরি করে মতবাদ খণ্ডন করেন:

```
NH₄CNO (অ্যামোনিয়াম সায়ানেট) → CO(NH₂)₂ (ইউরিয়া)
অজৈব যৌগ             তাপ      জৈব যৌগ
```

> 🔴 **মনে রাখো:** ওয়েলার = প্রথম পরীক্ষাগারে জৈব যৌগ তৈরি = প্রাণশক্তি মতবাদ ভুল প্রমাণ

## 🧬 কার্বনের বিশেষ বৈশিষ্ট্য — কেন এত বেশি জৈব যৌগ?

১. **শৃঙ্খলায়ন (Catenation):** কার্বন নিজে নিজের সাথে শিকল গঠন করতে পারে ২. **চতুর্যোজ্যতা:** কার্বনের যোজনী ৪ → নানা বন্ধন গঠনে সক্ষম ৩. **দ্বিবন্ধন ও ত্রিবন্ধন:** C=C এবং C≡C গঠনে সক্ষম ৪. **বৈচিত্র্যময় কার্বন কঙ্কাল:** সরল শিকল, শাখাযুক্ত শিকল, চক্রাকার শিকল

# 📚 সেকশন ২.১ — সমযোজী বন্ধন ও কার্বনের সংকরণ

## 🔥 হাই-ভ্যালু টপিক সতর্কতা

> ⚠️ **সংকরণ (Hybridization) সবচেয়ে গুরুত্বপূর্ণ অধ্যায়!** HSC, মেডিকেল ও ভার্সিটিতে প্রতি বছর এখান থেকে বহু প্রশ্ন আসে। **sp³, sp², sp সংকরণের পার্থক্য, বন্ধন কোণ, দূরত্ব সব মুখস্থ করতে হবে।**

## 📌 ২.১.২ সমযোজী বন্ধনের ইলেকট্রনীয় তত্ত্ব

- **১৯১৬ সালে লুইস** প্রকাশ করেন
- দুটি পরমাণু প্রত্যেকের নিকটতম নিষ্ক্রিয় গ্যাসের ইলেকট্রন বিন্যাস পেতে ইলেকট্রন জোড় শেয়ার করে
- সৃষ্ট বন্ধনকে **সমযোজী বন্ধন** বা **বন্ধন ইলেকট্রন জোড়** বলে
- কার্বনের উত্তেজিত অবস্থা: `C*(6) → 1s² [2s¹2pₓ¹2p_y¹2p_z¹]`

## 🔬 সিগমা (σ) ও পাই (π) বন্ধন

|বৈশিষ্ট্য|σ বন্ধন|π বন্ধন|
|---|---|---|
|**অক্ষ বরাবর অধিক্রমণ**|সামনে (head-on)|পাশাপাশি (side-on)|
|**শক্তি**|বেশি শক্তিশালী|কম শক্তিশালী|
|**ঘূর্ণন**|স্বাধীনভাবে ঘোরে|ঘুরতে পারে না|
|**একক বন্ধন**|১টি σ|নেই|
|**দ্বিবন্ধন**|১টি σ + ১টি π|১টি π|
|**ত্রিবন্ধন**|১টি σ + ২টি π|২টি π|

## 🧬 কার্বনের সংকরণ — বিস্তারিত তুলনামূলক সারণি

### ✅ sp³ সংকরণ (আলকেন / মিথেন CH₄, ইথেন C₂H₆)

```
উত্তেজিত C → 2s¹2pₓ¹2p_y¹2p_z¹
        sp³ সংকরণ → 4টি সমতুল্য sp³ অরবিটাল
```

|বৈশিষ্ট্য|মান|
|---|---|
|**অরবিটালের প্রকৃতি**|4টি sp³ অরবিটাল|
|**s-চরিত্র**|২৫%|
|**p-চরিত্র**|৭৫%|
|**জ্যামিতি**|চতুস্তলীয় (Tetrahedral)|
|**বন্ধন কোণ**|১০৯°২৮' (≈ ১০৯.৫°)|
|**C–C দূরত্ব (ইথেন)**|০.১৫৪ nm|
|**C–H দূরত্ব**|০.১০৯ nm|
|**উদাহরণ**|CH₄, C₂H₆, সব আলকেন|

### ✅ sp² সংকরণ (আলকিন / ইথিন C₂H₄)

```
উত্তেজিত C → *C(6) → 1s²[2s¹2pₓ¹2p_y¹]2p_z¹
        sp² সংকরণ → 3টি সমতলীয় sp² অরবিটাল + 1টি অপরিবর্তিত 2pz অরবিটাল
```

|বৈশিষ্ট্য|মান|
|---|---|
|**অরবিটালের প্রকৃতি**|3টি sp² + 1টি p|
|**s-চরিত্র**|৩৩.৩%|
|**p-চরিত্র**|৬৬.৭%|
|**জ্যামিতি**|সমতলীয় ত্রিকোণাকার|
|**বন্ধন কোণ**|১২০°|
|**C=C দূরত্ব (ইথিন)**|০.১৩৪ nm|
|**বন্ধন**|১টি σ + ১টি π (দ্বিবন্ধন)|
|**উদাহরণ**|C₂H₄, সব আলকিন, বেনজিন|

### ✅ sp সংকরণ (আলকাইন / ইথাইন C₂H₂)

```
উত্তেজিত C → 2s¹2pₓ¹ সংকরিত হয়
        sp সংকরণ → 2টি sp অরবিটাল + 2টি অপরিবর্তিত p অরবিটাল (2py ও 2pz)
```

|বৈশিষ্ট্য|মান|
|---|---|
|**অরবিটালের প্রকৃতি**|2টি sp + 2টি p|
|**s-চরিত্র**|৫০%|
|**p-চরিত্র**|৫০%|
|**জ্যামিতি**|রৈখিক (Linear)|
|**বন্ধন কোণ**|১৮০°|
|**C≡C দূরত্ব (ইথাইন)**|০.১২০ nm|
|**বন্ধন**|১টি σ + ২টি π (ত্রিবন্ধন)|
|**উদাহরণ**|C₂H₂ (অ্যাসিটিলিন), সব আলকাইন|

## 🚀 Quick Memory Trick — সংকরণ তুলনা

```
বন্ধন কোণ:  sp³ (109.5°) < sp² (120°) < sp (180°)
C-C দূরত্ব: sp (120 pm) < sp² (134 pm) < sp³ (154 pm)
s-চরিত্র:   sp³ (25%) < sp² (33.3%) < sp (50%)
অ্যাসিডিটি: sp³-H < sp²-H < sp-H (s-চরিত্র বাড়লে অ্যাসিডিটি বাড়ে)
```

### IUPAC নামকরণের ধাপ (Strategy)

১. **দীর্ঘতম কার্বন শিকল** খুঁজে বের করো (মূল শিকল) 
২. কার্যকরী মূলক বা দ্বিবন্ধন/ত্রিবন্ধনকে **সর্বনিম্ন সংখ্যা** দাও 
৩. শাখা বা প্রতিস্থাপক মূলকগুলোকে বর্ণানুক্রমে সাজাও 
৪. প্রতিটি C-পরমাণুতে বাকি যোজনী H পরমাণু যোগ করো

# 📚 সেকশন ২.৩ — সমাণুতা (Isomerism)

# ═══════════════════════════════════════════

## 🔥 হাই-ভ্যালু টপিক সতর্কতা

> ⚠️ **সমাণুতা মেডিকেল ও ভার্সিটি পরীক্ষায় সবচেয়ে বেশি আসে!** 
> বিশেষত জ্যামিতিক সমাণু (cis-trans), আলোক সমাণু (enantiomers, diastereomers), কাইরাল কেন্দ্র — এগুলো খুব ভালোভাবে বুঝে পড়তে হবে।

## 📌 সমাণুতার প্রকারভেদ

```
সমাণুতা
├── কাঠামোগত বা গঠন সমাণুতা (Structural Isomerism)
│   ├── শিকল সমাণুতা (Chain Isomerism)
│   ├── স্থান সমাণুতা (Position Isomerism)
│   ├── কার্যকরী মূলক সমাণুতা (Functional Group Isomerism)
│   └── মেটামেরিজম (Metamerism)
└── স্থানিক সমাণুতা (Stereo Isomerism)
    ├── জ্যামিতিক সমাণুতা (Geometric Isomerism) [cis-trans]
    └── আলোক সমাণুতা (Optical Isomerism)
        ├── এনানসিওমার (Enantiomers)
        └── ডায়াস্টিরিওমার (Diastereomers)
```

---

## 🔬 জ্যামিতিক সমাণুতা (Geometric / cis-trans Isomerism)

### শর্ত:

- দ্বিবন্ধনযুক্ত দুটি C পরমাণুতে প্রতিটিতে ভিন্ন পরমাণু বা মূলক থাকতে হবে
- abC=Ccd সাধারণ সংকেত অনুসরণ করে

|বৈশিষ্ট্য|সিস-সমাণু (cis)|ট্রান্স-সমাণু (trans)|
|---|---|---|
|**একই মূলক**|একই দিকে|বিপরীত দিকে|
|**গলনাঙ্ক**|সাধারণত কম|সাধারণত বেশি|
|**সুষমতা**|অধিক সুষম|কম সুষম|
|**দ্রবণীয়তা**|পানিতে বেশি|পানিতে কম|
|**ঘনত্ব**|বেশি|কম|
|**উদাহরণ**|সিস-বিউট-২-ইন (b.p 4°C)|ট্রান্স-বিউট-২-ইন (b.p 1°C)|

> 🔴 **গুরুত্বপূর্ণ:** ম্যালেয়িক এসিড (সিস) ও ফিউমারিক এসিড (ট্রান্স) — সিস সমাণুর গলনাঙ্ক ১৩৫°C, ট্রান্সের ২৮৭°C

---

## 🔬 আলোক সমাণুতা (Optical Isomerism)

### কাইরাল কেন্দ্র (Chiral Centre):

- যে C পরমাণুতে **চারটি ভিন্ন পরমাণু বা মূলক** যুক্ত থাকে তাকে **অপ্রতিসম বা কাইরাল কার্বন পরমাণু** বলে
- তারকা (*) চিহ্ন দিয়ে চিহ্নিত করা হয়

### আলোক সমাণুর সংখ্যা নির্ণয়:

|অপ্রতিসম C-পরমাণু (n)|আলোক সমাণুর সংখ্যা|
|---|---|
|1|2¹ = ২ (১টি d, ১টি l)|
|2|2² = ৪ (কিন্তু মেসো থাকলে কম)|
|3|2³ = ৮|
|n|2ⁿ|

### এনানসিওমার বনাম ডায়াস্টিরিওমার:

|বৈশিষ্ট্য|এনানসিওমার|ডায়াস্টিরিওমার|
|---|---|---|
|**সম্পর্ক**|পরস্পরের দর্পণ প্রতিবিম্ব|দর্পণ প্রতিবিম্ব নয়|
|**ভৌত ধর্ম**|একই (গলনাঙ্ক, স্ফুটনাঙ্ক)|ভিন্ন|
|**আলোর ঘূর্ণন**|বিপরীত দিকে|ভিন্নভাবে|

# 📚 সেকশন ২.৪ — পলিমারাইজেশন (Polymerization)

# ═══════════════════════════════════════════

## 🔥 হাই-ভ্যালু টপিক সতর্কতা

> ⚠️ **পলিমারাইজেশন থেকে MCQ ও সংক্ষিপ্ত প্রশ্ন প্রায়ই আসে।** পলিথিন তৈরির শর্ত (তাপ, চাপ, প্রভাবক) মনে রাখা জরুরি।

## 📌 পলিমারাইজেশনের প্রকারভেদ

### ১. সংযোজন পলিমারাইজেশন (Addition Polymerization):

- একই মনোমার বারবার সংযুক্ত হয়ে বৃহৎ পলিমার তৈরি করে
- **শর্ত:** তাপ (৭০°C), চাপ (৫–১০ atm), প্রভাবক (Cr₂O₃)
- মনোমারে দ্বিবন্ধন (>C=C<) থাকে

```
n(CH₂=CH₂) → [–CH₂–CH₂–]ₙ
 ইথিলিন        পলিথিন
(মনোমার)      (পলিমার)
উচ্চ তাপ, ২০০°C, প্রভাবক O₂
```

### ২. ঘনীভবন পলিমারাইজেশন (Condensation Polymerization):

- দুটি ভিন্ন মনোমারের বিক্রিয়ায় ছোট অণু (H₂O, HCl ইত্যাদি) বেরিয়ে যায়
- উদাহরণ: নাইলন-৬,৬, ডেক্রন, বেকেলাইট

---

## 📊 গুরুত্বপূর্ণ পলিমারসমূহ

|পলিমারের নাম|মনোমার|বিক্রিয়ার ধরন|ব্যবহার|
|---|---|---|---|
|**পলিথিন**|ইথিলিন (CH₂=CH₂)|সংযোজন|ব্যাগ, বোতল|
|**পলিভিনাইল ক্লোরাইড (PVC)**|ভিনাইল ক্লোরাইড|সংযোজন|পাইপ, তার|
|**পলিস্টাইরিন**|স্টাইরিন|সংযোজন|প্যাকেজিং|
|**টেফলন (PTFE)**|টেট্রাফ্লুরোইথিলিন|সংযোজন|নন-স্টিক কুকওয়্যার|

# 📚 সেকশন ২.৫ — আলকেন (Alkane)

# ═══════════════════════════════════════════

## 🔥 হাই-ভ্যালু টপিক সতর্কতা

> ⚠️ **আলকেনের বিক্রিয়াসমূহ (বিশেষত হ্যালোজেনেশন) পরীক্ষায় বেশি আসে।** ফ্রি-র‍্যাডিকেল প্রতিস্থাপন মেকানিজম ধাপে ধাপে লিখতে পারতে হবে।

## 📌 আলকেনের বিক্রিয়াসমূহ

### ১. দহন বিক্রিয়া (Combustion):

```
CH₄ + 2O₂ → CO₂ + 2H₂O   [ΔH° = –802 kJ.mol⁻¹]
পূর্ণ দহন: C₁₁H₂₄(l) + 17O₂(g) → 11CO₂(g) + 12H₂O(g)
```

### ২. হ্যালোজেনেশন (প্রতিস্থাপন) — **ফ্রি-র‍্যাডিকেল মেকানিজম:**

```
CH₄ + Cl₂ → CH₃Cl + HCl  [UV আলো, ২৫°C]
```

**তিনটি ধাপ:**

|ধাপ|নাম|বিক্রিয়া|
|---|---|---|
|**১ম**|শুরু (Initiation)|Cl₂ → 2Cl• (UV আলোয়)|
|**২য়**|প্রসার (Propagation)|CH₄ + Cl• → CH₃• + HCl; CH₃• + Cl₂ → CH₃Cl + Cl•|
|**৩য়**|সমাপ্তি (Termination)|Cl• + Cl• → Cl₂; CH₃• + Cl• → CH₃Cl|

> 🔴 **হ্যালোজেনের সক্রিয়তার ক্রম:** F₂ > Cl₂ > Br₂ > I₂ 🔴 **H-পরমাণুর সক্রিয়তার ক্রম:** 3°H > 2°H > 1°H

---

## 📊 আলকেনের শনাক্তকারী বিক্রিয়া:

আলকেন কম সক্রিয় → তাই একে **প্যারাফিন** বলে

- অ্যাসিড, ক্ষার, সাধারণ বিকারকের সাথে বিক্রিয়া করে না
- শুধু দহন ও হ্যালোজেনেশন ঘটে

# 📚 সেকশন ২.৬ — আলকিন (Alkene)

# ═══════════════════════════════════════════

## 🔥 হাই-ভ্যালু টপিক সতর্কতা

> ⚠️ **আলকিনের বিক্রিয়া এবং মারকনিকভের নিয়ম মেডিকেলের জন্য অতি গুরুত্বপূর্ণ!** এছাড়া আরোমেটিসিটি ও হাকেল নিয়ম বুঝতে না পারলে বেনজিন বোঝা যাবে না।

## 📌 আলকিনের বিক্রিয়াসমূহ

### (i) হাইড্রোজেনেশন (সংযোজন বিক্রিয়া):

```
CH₂=CH₂ + H₂ → CH₃–CH₃  [Pd/BaSO₄, তাপ]
ইথিলিন            ইথেন
```

### (ii) হ্যালোজেনেশন:

```
CH₂=CH₂ + Br₂ → CH₂Br–CH₂Br  [CCl₄ দ্রাবক]
ইথিলিন   ব্রোমিন    1,2-ডাইব্রোমোইথেন
```

> 🔴 **ব্রোমিন পানির লাল-বাদামি রং বিবর্ণ হয় → আলকিন শনাক্তকরণ পরীক্ষা**

### (iii) HX সংযোজন — মারকনিকভের নিয়ম:

**নিয়ম:** অপ্রতিসম আলকিনে HX সংযোজনে H পরমাণু সেই C-পরমাণুতে যুক্ত হয় যেখানে ইতিমধ্যে বেশি H আছে।

```
CH₃–CH=CH₂ + HBr → CH₃–CHBr–CH₃  (প্রধান উৎপাদ, ২°-হ্যালাইড)
                                  + CH₃–CH₂–CH₂Br  (অপ্রধান, ১°-হ্যালাইড)
```

> 🔴 **বিপরীত মারকনিকভের নিয়ম (খারাসের নিয়ম):** জৈব পারঅক্সাইডের উপস্থিতিতে HBr সংযোজনে বিপরীত ঘটে — শুধু HBr-এর বেলায়, HCl ও HI-এর বেলায় নয়

### (iv) H₂SO₄ সহ আলকিনের বিক্রিয়া:

```
CH₂=CH₂ + H–OSO₃H → CH₂–CH₂–OSO₃H  [H₂O] → CH₃CH₂OH + H₂SO₄
ইথিন                  ইথাইল হাইড্রোজেন সালফেট     ইথানল
```

মারকনিকভের নিয়ম প্রযোজ্য।

### (v) ওজোনোলাইসিস:

```
R–CH=CH–R' + O₃/CCl₄ → [Ozonide] → R–CHO + R'–CHO  [Zn/H₂O]
```

- দ্বিবন্ধনযুক্ত C-পরমাণুতে H থাকলে → আলডিহাইড
- দ্বিবন্ধনযুক্ত C-পরমাণুতে H না থাকলে → কিটোন

---

## 📊 আলকিন শনাক্তকরণ

|পরীক্ষা|ফলাফল|
|---|---|
|ব্রোমিন পানি (লাল-বাদামি)|বিবর্ণ হয় → আলকিন উপস্থিত|
|KMnO₄ (গাঢ় বেগুনি)|বিবর্ণ হয় → দ্বিবন্ধন উপস্থিত|
|Baeyer পরীক্ষা|KMnO₄ বিবর্ণ|

## 🔬 আরোমেটিসিটি ও হাকেল নিয়ম

### হাকেল নিয়মের শর্তসমূহ:

১. ✅ যৌগটির গঠন **চক্রীয় ও সমতলীয়** হতে হবে ২. ✅ বলয় গঠনকারী প্রতিটি পরমাণুতে **p-অরবিটাল** থাকতে হবে ৩. ✅ সঞ্চরণশীল π ইলেকট্রন সংখ্যা **(4n + 2)** হতে হবে, যেখানে n = 0, 1, 2, 3...

|n মান|π ইলেকট্রন সংখ্যা (4n+2)|উদাহরণ|
|---|---|---|
|0|2|ত্রিচক্রাকার ক্যাটায়ন|
|1|**6**|**বেনজিন (C₆H₆)** ← সবচেয়ে গুরুত্বপূর্ণ|
|2|10|নেপথালিন|
|3|14|অ্যান্থ্রাসিন|

> 🔴 **বেনজিনে 6টি π ইলেকট্রন → আরোমেটিক (হাকেল নিয়ম মেনে চলে)**

# 📚 সেকশন ২.৭ — আলকাইন (Alkyne)

# ═══════════════════════════════════════════

## 🔥 হাই-ভ্যালু টপিক সতর্কতা

> ⚠️ **আলকাইনের অ্যাসিডিক ধর্ম** (টার্মিনাল আলকাইন) এবং **অ্যাসিটিলিনের শিল্পসম্মত উৎপাদন** পরীক্ষায় আসে।

## 📌 আলকাইনের বিশেষ ধর্ম

### অ্যাসিডিক ধর্ম (sp সংকরণের কারণে):

```
HC≡CH + NaNH₂ → HC≡C–Na⁺ + NH₃  [টার্মিনাল আলকাইন]
```

- sp-কার্বনের s-চরিত্র ৫০% → H পরমাণু বেশি দৃঢ়ভাবে আকর্ষণ → সহজে H⁺ ত্যাগ করে
- **অ্যাসিডিটি:** আলকাইন > আলকিন > আলকেন

### হাইড্রেশন বিক্রিয়া (Wacker Process):

```
HC≡CH + H₂O → [CH₂=CHOH] → CH₃CHO  [H₂SO₄, HgSO₄, ৬০°C]
অ্যাসিটিলিন   ভিনাইল অ্যালকোহল  অ্যাসিটালডিহাইড
```

# 📚 সেকশন ২.৮ — বেনজিন ও অ্যারোমেটিক যৌগ

# ═══════════════════════════════════════════

## 🔥 হাই-ভ্যালু টপিক সতর্কতা

> ⚠️ **বেনজিনের ইলেকট্রোফিলিক প্রতিস্থাপন বিক্রিয়া (নাইট্রেশন, হ্যালোজেনেশন, ফ্রিডেল-ক্রাফট) মেডিকেল ও ভার্সিটিতে সর্বাধিক আসে।** AlCl₃ কেন ব্যবহার করা হয় এটা বুঝে রাখতে হবে।

## 📌 বেনজিনের ইলেকট্রোফিলিক প্রতিস্থাপন বিক্রিয়া

### ১. নাইট্রেশন (Nitration):

```
C₆H₆ + HNO₃ → C₆H₅–NO₂ + H₂O  [গাঢ় H₂SO₄, ৫০°C]
বেনজিন  নাইট্রিক এসিড  নাইট্রোবেনজিন
```

- ইলেকট্রোফাইল: NO₂⁺ (নাইট্রোনিয়াম আয়ন)
- H₂SO₄ প্রভাবকরূপে কাজ করে

### ২. হ্যালোজেনেশন:

```
C₆H₆ + Cl₂ → C₆H₅Cl + HCl  [অনার্দ্র AlCl₃, ৮০°C]
বেনজিন         ক্লোরোবেনজিন
```

### ৩. ফ্রিডেল-ক্রাফট অ্যালকাইলেশন:

```
C₆H₆ + CH₃Cl → C₆H₅CH₃ + HCl  [অনার্দ্র AlCl₃]
বেনজিন           টলুইন
```

### ৪. ফ্রিডেল-ক্রাফট অ্যাসাইলেশন:

```
C₆H₆ + CH₃COCl → C₆H₅COCH₃ + HCl  [অনার্দ্র AlCl₃]
                   অ্যাসিটোফেনোন
```

> 🔴 **AlCl₃ কেন ব্যবহার করা হয়?** AlCl₃ লুইস এসিড। এটি R-X, RCO-X বা X₂ থেকে ইলেকট্রোফাইল (R⁺, RCO⁺, Cl⁺) তৈরি করে, যা বেনজিন বলয়ে আক্রমণ করে।

### ৫. সালফোনেশন:

```
C₆H₆ + H₂SO₄ (ধূমায়িত) → C₆H₅–SO₃H + H₂O  [উত্তাপ]
                            বেনজিন সালফোনিক এসিড
```

---

## 📌 বেনজিনের প্রতিস্থাপক প্রভাব

|মূলকের প্রকার|উদাহরণ|অর্থো-প্যারা নির্দেশক|মেটা নির্দেশক|
|---|---|---|---|
|**ইলেকট্রন দাতা**|–OH, –NH₂, –CH₃, –OCH₃|✅ হ্যাঁ (o, p)||
|**ইলেকট্রন গ্রাহক**|–NO₂, –CHO, –COOH, –SO₃H||✅ হ্যাঁ (m)|

# 📚 সেকশন ২.৯ — হ্যালোআলকেন ও হ্যালোঅ্যারিন

# ═══════════════════════════════════════════

## 🔥 হাই-ভ্যালু টপিক সতর্কতা

> ⚠️ **নিউক্লিওফিলিক প্রতিস্থাপন বিক্রিয়া (SN1 ও SN2) এবং অপসারণ বিক্রিয়া (E1 ও E2)** — এই মেকানিজম দুটি বোর্ড ও মেডিকেলে সরাসরি আসে। পার্থক্য বুঝতেই হবে।

## 📌 নিউক্লিওফিলিক প্রতিস্থাপন বিক্রিয়া

### SN1 বিক্রিয়া (একআণবিক):

- **২টি ধাপে** ঘটে → প্রথমে কার্বোনিয়াম আয়ন তৈরি, তারপর নিউক্লিওফাইল আক্রমণ
- **টার্সিয়ারি (3°) হ্যালাইড** বেশি পছন্দ করে
- **পোলার প্রোটিক দ্রাবক** (পানি, অ্যালকোহল) অনুকূল
- Racemization ঘটে (R ও S মিশ্রণ)

```
(CH₃)₃C–Br → (CH₃)₃C⁺ + Br⁻  [ধীরে, প্রথম ধাপ]
             ↓ Nu⁻
          (CH₃)₃C–Nu [দ্রুত, দ্বিতীয় ধাপ]
```

### SN2 বিক্রিয়া (দ্বিআণবিক):

- **১টি ধাপে** ঘটে → নিউক্লিওফাইল পেছন থেকে আক্রমণ করে একই সাথে হ্যালাইড বের হয়
- **প্রাইমারি (1°) হ্যালাইড** বেশি পছন্দ করে
- **পোলার অ্যাপ্রোটিক দ্রাবক** (DMSO, অ্যাসিটোন) অনুকূল
- Inversion of configuration (Walden inversion) ঘটে

```
Nu⁻ + R–X → Nu---R---X → Nu–R + X⁻  [এক ধাপে]
```

### SN1 ও SN2 তুলনামূলক সারণি:

|বৈশিষ্ট্য|SN1|SN2|
|---|---|---|
|**ধাপ সংখ্যা**|২|১|
|**হ্যালাইডের ধরন**|3° > 2° > 1°|1° > 2° > 3°|
|**মধ্যবর্তী যৌগ**|কার্বোনিয়াম আয়ন|কোনোটি নেই|
|**গতিবিধি বিন্যাস পরিবর্তন**|রেসিমাইজেশন|ইনভার্শন|
|**দ্রাবক**|পোলার প্রোটিক|পোলার অ্যাপ্রোটিক|

---

## 📌 অপসারণ বিক্রিয়া (Elimination)

### E2 বিক্রিয়া (অ্যালকোহলীয় KOH):

```
CH₃CH₂Br + KOH (alc.) → CH₂=CH₂ + KBr + H₂O
```

- β-কার্বন থেকে H এবং α-কার্বন থেকে Br একসাথে বের হয় → আলকিন উৎপন্ন

### E1 বিক্রিয়া:

```
(CH₃)₃C–Br → (CH₃)₃C⁺ → (CH₃)₂C=CH₂ + H⁺
```

### প্রতিস্থাপন বনাম অপসারণ:

- **হ্যালোজেনো আলকেন যত টার্সিয়ারি** → E2 মেকানিজমে বেশি ঘটে
- **প্রাইমারি** → SN2 মেকানিজম প্রাধান্য পায়

---

## 📌 ক্লোরোফর্ম (CHCl₃) — বিশেষ গুরুত্বপূর্ণ

|বিষয়|তথ্য|
|---|---|
|**IUPAC নাম**|ট্রাইক্লোরোমিথেন|
|**সংকেত**|CHCl₃|
|**ব্যবহার**|চেতনানাশক (সার্জারিতে), শিল্পক্ষেত্রে দ্রাবক|
|**বিষক্রিয়া**|সূর্যালোক ও বাতাসের সংস্পর্শে ফসজিন (COCl₂) গ্যাস তৈরি|
|**সংরক্ষণ**|বাদামি বর্ণের বোতলে, আলো আসতে দেওয়া হয় না; ১% ইথানল মিশানো হয়|

### ক্লোরোফর্মের বিশুদ্ধতা পরীক্ষা:

```
2CHCl₃ + O₂ (আলো) → 2COCl₂ + 2HCl
HCl + AgNO₃ (aq) → AgCl (সাদা অধঃক্ষেপ) + HNO₃
বিশুদ্ধ ক্লোরোফর্ম AgNO₃ দ্রবণের সাথে কোনো অধঃক্ষেপ দেয় না।
```

# 📚 সেকশন ২.১০ — অ্যালকোহল, ইথার ও গ্লিসারিন

# ═══════════════════════════════════════════

## 🔥 হাই-ভ্যালু টপিক সতর্কতা

> ⚠️ **অ্যালকোহলের শ্রেণিবিভাগ, শনাক্তকারী বিক্রিয়া, ইথানলের প্রস্তুতি** — এগুলো পরীক্ষায় সরাসরি আসে। মিথানলের বিষাক্ততা ও আয়োডোফর্ম পরীক্ষা মনে রাখা বাধ্যতামূলক।

## 📌 অ্যালকোহলের শ্রেণিবিভাগ

|শ্রেণি|সংজ্ঞা|উদাহরণ|
|---|---|---|
|**প্রাইমারি (1°)**|–OH যুক্ত C-এ ১টি অ্যালকাইল মূলক|CH₃CH₂OH (ইথানল)|
|**সেকেন্ডারি (2°)**|–OH যুক্ত C-এ ২টি অ্যালকাইল মূলক|(CH₃)₂CHOH (প্রোপানল-2)|
|**টার্সিয়ারি (3°)**|–OH যুক্ C-এ ৩টি অ্যালকাইল মূলক|(CH₃)₃COH (tert-বিউটানল)|

---

## 📌 অ্যালকোহলের শনাক্তকারী বিক্রিয়া

### ১. ধাতব Na সহ পরীক্ষা:

```
2R–OH + 2Na → 2R–ONa + H₂↑
2C₂H₅OH + 2Na → 2C₂H₅ONa + H₂↑
```

→ H₂ গ্যাস উৎপন্ন হলে অ্যালকোহলিক –OH মূলক প্রমাণ

### ২. PCl₅ সহ পরীক্ষা:

```
R–OH + PCl₅ → R–Cl + POCl₃ + HCl(g)
HCl(g) + NH₃(g) → NH₄Cl (সাদা ধোঁয়া)
```

→ NH₃ দ্রবণ সিক্ত কাচের রড ধরলে সাদা ধোঁয়া → অ্যালকোহল প্রমাণ

### ৩. আয়োডোফর্ম পরীক্ষা (ইথানল শনাক্তকরণ):

```
CH₃CH₂OH + I₂ + NaOH → CHI₃↓ + HCOONa + NaI + H₂O
                         আয়োডোফর্ম (হলুদ অধঃক্ষেপ)
```

> 🔴 **আয়োডোফর্ম বা হ্যালোফর্ম পরীক্ষার শর্ত:** মিথাইল কার্বনিল মূলক (CH₃CO–) যুক্ত যেকোনো কার্বনিল যৌগ বা অ্যালকোহল যা হ্যালোজেন দ্বারা জারিত হয়ে CH₃CO– মূলকযুক্ত কার্বনিল যৌগে পরিণত হয় তারাই হ্যালোফর্ম বিক্রিয়া দেয়।

---

## 📌 মিথানল (Wood Spirit) — বিষাক্ত!

- কাঠের অস্থায়িক পাতনে (৩৫০°C) পাইরোলিগনাস এসিড মিশ্রণ থেকে পাওয়া যায়
- উড-স্পিরিটে **৯৮% মিথানল ও ১-২% অ্যাসিটোন** মিশ্রিত থাকে
- স্ফুটনাঙ্ক **৬৫°C**
- মিথানল পান করলে → অন্ধত্ব, মৃত্যু (অক্সিডেশনে ফর্মিক এসিড ও ফর্মালডিহাইড তৈরি হয় যা চোখের স্নায়ু নষ্ট করে)

---

## 📌 গ্লিসারিন (Glycerine/Glycerol)

**সংকেত:** HOCH₂–CH(OH)–CH₂OH (প্রোপেন-১,২,৩-ট্রাইঅল) **অপর নাম:** গ্লিসারল বা প্রোপেন-১,২,৩-ট্রাইঅল

### প্রস্তুতি পদ্ধতি:

#### ১. তেল বা চর্বির ক্ষারীয় আর্দ্র বিশ্লেষণ (সাবানায়ন):

```
CH₂–O.OCR           CH₂–OH
|                    |
CH–O.OCR + 3NaOH → CH–OH + 3R–COONa
|                    |
CH₂–O.OCR           CH₂–OH
তেল বা চর্বি          গ্লিসারিন   সাবান
```

#### ২. তেল বা চর্বির অম্লীয় আর্দ্র বিশ্লেষণ:

```
তেল + 3H–OH → গ্লিসারিন + ফ্যাটি এসিড  [H⁺, তাপ]
```

#### ৩. প্রোপিন থেকে গ্লিসারিন (আধুনিক সংশ্লেষণ)

### গ্লিসারিনের ব্যবহার:

- সাবান শিল্পে উপজাত হিসেবে পাওয়া যায়
- ওষুধ শিল্পে (ত্বক নরম করতে)
- বিস্ফোরক: নাইট্রোগ্লিসারিন তৈরিতে
- মিষ্টান্ন শিল্পে (হাইগ্রোস্কোপিক ধর্মের কারণে)

# 📚 সেকশন ২.১১ — কার্বনিল যৌগ: আলডিহাইড ও কিটোন

# ═══════════════════════════════════════════

## 🔥 হাই-ভ্যালু টপিক সতর্কতা

> ⚠️ **আলডিহাইড ও কিটোনের পার্থক্যকারী পরীক্ষা (টলেন, ফেলিং, 2,4-DNP) বোর্ড ও মেডিকেলে বারবার আসে।** ক্যানিজারো বিক্রিয়া ও অ্যালডল ঘনীভবন মনে রাখতে হবে।

## 📌 আলডিহাইড ও কিটোনের পার্থক্য

|বৈশিষ্ট্য|আলডিহাইড (R–CHO)|কিটোন (R–CO–R')|
|---|---|---|
|**মূলক**|–CHO (কার্বন শিকলের শেষে)|–CO– (মাঝে)|
|**সাধারণ সংকেত**|CₙH₂ₙO|CₙH₂ₙO|
|**টলেন বিকারক**|রুপার আয়না (Ag) তৈরি ✅|বিক্রিয়া করে না ❌|
|**ফেলিং দ্রবণ**|লাল অধঃক্ষেপ (Cu₂O) ✅|বিক্রিয়া করে না ❌|
|**2,4-DNP পরীক্ষা**|হলুদ অধঃক্ষেপ ✅|হলুদ অধঃক্ষেপ ✅|
|**KMnO₄**|বিবর্ণ করে ✅|পরিস্থিতি সাপেক্ষ|

---

## 📌 গুরুত্বপূর্ণ বিকারক ও পরীক্ষাসমূহ

### টলেন বিকারক (Tollens' Reagent):

- **গঠন:** AgNO₃ + NH₄OH → [Ag(NH₃)₂]OH (ডায়ামিন সিলভার হাইড্রোক্সাইড)
- আলডিহাইড → চকচকে রুপার আয়না (রুপার দর্পণ পরীক্ষা)

```
R–CHO + 2[Ag(NH₃)₂]OH → R–COONH₄ + 2Ag↓ + 3NH₃ + H₂O
                          রুপার আয়না
```

### ফেলিং দ্রবণ (Fehling's Solution):

- **গঠন:** ফেলিং A (CuSO₄) + ফেলিং B (NaOH + সোডিয়াম পটাসিয়াম টার্ট্রেট)
- আলডিহাইড → ইট-লাল অধঃক্ষেপ (Cu₂O)

```
R–CHO + 2Cu²⁺ + 5OH⁻ → R–COO⁻ + Cu₂O↓ + 3H₂O
                        লাল অধঃক্ষেপ
```

### 2,4-DNP পরীক্ষা:

- **উভয়** আলডিহাইড ও কিটোন → হলুদ বা কমলা অধঃক্ষেপ
- কার্বনিল যৌগ শনাক্তের সাধারণ পরীক্ষা

---

## 📌 বিশেষ বিক্রিয়াসমূহ

### ক্যানিজারো বিক্রিয়া (Cannizzaro Reaction):

- **শর্ত:** α-H নেই এমন আলডিহাইড + গাঢ় NaOH
- একটি অণু জারিত হয় (→ এসিড), অন্যটি বিজারিত হয় (→ অ্যালকোহল)

```
2HCHO + NaOH → CH₃OH + HCOONa
2C₆H₅CHO + NaOH → C₆H₅CH₂OH + C₆H₅COONa
```

### অ্যালডল ঘনীভবন (Aldol Condensation):

- **শর্ত:** α-H যুক্ত আলডিহাইড + তনু ক্ষার

```
2CH₃CHO → CH₃CH(OH)CH₂CHO → CH₃CH=CHCHO + H₂O
ইথান্যাল   3-হাইড্রোক্সিবিউটান্যাল   ক্রোটোনালডিহাইড
```

## 🔥 হাই-ভ্যালু টপিক সতর্কতা

|শ্রেণি|বর্ণনা|উদাহরণ|

```
R–NH₂ + CHCl₃ + 3KOH → R–N≡C + 3KCl + 3H₂O
(1° অ্যামিন)            কার্বিলঅ্যামিন (দুর্গন্ধ)
```

- শুধু **প্রাইমারি অ্যামিন** এ বিক্রিয়া দেয়

> ⚠️ **গ্লুকোজ ও ফ্রুক্টোজের গঠন, গ্লাইকোসাইড বন্ধন এবং পলিস্যাকারাইড** মেডিকেল পরীক্ষায় বেশি আসে।

## 📌 কার্বোহাইড্রেটের শ্রেণিবিভাগ:

```
কার্বোহাইড্রেট
├── মনোস্যাকারাইড (একক শর্করা): গ্লুকোজ, ফ্রুক্টোজ, গ্যালাকটোজ
├── ডাইস্যাকারাইড (দ্বি শর্করা): সুক্রোজ, মল্টোজ, ল্যাকটোজ
└── পলিস্যাকারাইড (বহু শর্করা): স্টার্চ, গ্লাইকোজেন, সেলুলোজ
```

## 📌 গ্লাইকোসাইড বন্ধন:

- মনোস্যাকারাইডের হেমিঅ্যাসিটাল –OH মূলক ও অন্য শর্করার –OH মূলকের মধ্যে পানি অপসারণে গঠিত
- **১,৪-গ্লাইকোসাইড বন্ধন:** স্টার্চ ও সেলুলোজে (α ও β উভয়)
- **α-১,৪ গ্লাইকোসাইড বন্ধন:** স্টার্চ ও গ্লাইকোজেনে (শরীরে হজম হয়)
- **β-১,৪ গ্লাইকোসাইড বন্ধন:** সেলুলোজে (মানব শরীরে হজম হয় না)

## 📌 পেপটাইড বন্ধন:

- দুটি অ্যামিনো এসিডের মধ্যে পানি অপসারণে গঠিত

```
–COOH + H₂N– → –CO–NH– + H₂O
                পেপটাইড বন্ধন
```

## 🔴 সর্বোচ্চ গুরুত্বপূর্ণ ফ্যাক্টস (One-liners)

```
✅ ওয়েলার (১৮২৮) → প্রথম পরীক্ষাগারে জৈব যৌগ (ইউরিয়া) তৈরি → প্রাণশক্তি মতবাদ খণ্ডন
✅ sp³ → 109.5° → CH₄ → tetrahedral
✅ sp² → 120° → C₂H₄ → trigonal planar
✅ sp → 180° → C₂H₂ → linear
✅ হাকেল নিয়ম → 4n+2 π ইলেকট্রন → আরোমেটিক
✅ বেনজিনে 6 π ইলেকট্রন → n=1 → আরোমেটিক
✅ মারকনিকভের নিয়ম → H যায় যেখানে বেশি H আছে
✅ খারাসের নিয়ম → HBr মারকনিকভের বিপরীত (পারঅক্সাইড উপস্থিতিতে)
✅ SN1 → টার্সিয়ারি হ্যালাইড → carbocation intermediate
✅ SN2 → প্রাইমারি হ্যালাইড → এক ধাপে → inversion
✅ টলেন বিকারক → শুধু আলডিহাইড → রুপার আয়না
✅ ফেলিং দ্রবণ → শুধু আলডিহাইড → Cu₂O (লাল)
✅ 2,4-DNP → উভয় আলডিহাইড ও কিটোন → হলুদ অধঃক্ষেপ
✅ আয়োডোফর্ম পরীক্ষা → ইথানল ও মিথাইল কার্বনিল যৌগ → CHI₃ (হলুদ)
✅ ক্যানিজারো → α-H নেই এমন আলডিহাইড + গাঢ় NaOH → disproportionation
✅ IR-OH: 3200–3550 cm⁻¹; IR-C=O (কিটোন): 1715 cm⁻¹
✅ ডেটলের মূল উপাদান → ক্লোরোজাইলিনল (৪.৮%)
✅ পাইন তেলের মূল উপাদান → α-Terpineol (C₁₀H₁₈O)
✅ থার্মোপ্লাস্টিক → পুনর্ব্যবহারযোগ্য (যেমন পলিথিন)
✅ থার্মোসেট → পুনর্গলানো যায় না (যেমন বেকেলাইট)
✅ গ্লিসারিন প্রস্তুতি → তেল বা চর্বির সাবানায়ন
✅ β-১,৪ গ্লাইকোসাইড বন্ধন → সেলুলোজে → মানব শরীরে হজম হয় না
```

## ⚠️ সবচেয়ে বেশি ভুল হওয়া বিষয়সমূহ

```
❗ sp² সংকরণে শুধু ৩টি sp² অরবিটাল + ১টি অপরিবর্তিত p (ভুলে যাবে না!)
❗ C=C দূরত্ব: ইথেন (154 pm) > ইথিন (134 pm) > ইথাইন (120 pm)
❗ হ্যালোজেনের বিক্রিয়াশীলতা: F₂ > Cl₂ > Br₂ > I₂ (কিন্তু I₂ আলকিনের সাথে প্রায় বিক্রিয়া করে না)
❗ খারাসের নিয়ম শুধু HBr-এ, HCl ও HI-এ নয়!
❗ ক্লোরোফর্ম বিষাক্ত ফসজিন তৈরি করে সূর্যালোকে → তাই বাদামি বোতলে সংরক্ষণ
❗ মিথানল ও ইথানলের পার্থক্য: মিথানল মারাত্মক বিষাক্ত!
❗ α-H মানে কার্বনিল মূলকের পাশের কার্বনের H
```
