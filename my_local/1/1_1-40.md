#### 9. If the message bit is 111 then the Even-parity bit will be:  #os/parity  

(a) 1111 
(b) 1 
(c) 0 
(d) None 

Ans: b 

Note: A parity bit, also known as a check bit, is a single bit that can be appended to a binary string. It is set to either 1 or 0 to make the total number of ==1-bits either even ("even parity")== or ==odd ("odd parity").== 
- If the message already has an even number of **1s**, the parity bit is `0`.
- If the message has an odd number of **1s**, the parity bit is `1` to make the count even.
- **Example 1: Message = `10101`**
    
    - Count of `1s` = 3 (odd).
    - Parity bit = `1` (to make the count even).
    - Final Message: `101011`.
- **Example 2: Message = `10010`**
    
    - Count of `1s` = 2 (even).
    - Parity bit = `0` (since it's already even).
    - Final Message: `100100`.
